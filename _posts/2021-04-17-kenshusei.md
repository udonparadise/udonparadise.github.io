夜公演に参加。  
江端妃咲ちゃんがかわいくてびっくりした。  
無発声なのにめちゃめちゃ楽しかった。  
全曲研修生曲ってすごい。

# 彼女になりたいっ！！！

当たり前のようにある1番サビのえばちゃんソロ。  
オレンジ色のペンライトが揺れまくっていた。さすが人気メン。

# ありがた迷惑物語

初披露されたときから大好きな曲。  
ドロドロした歌詞を10代の女の子が歌うのが良いよね。

# 悪いヒト

明るい曲調だけど内容はTHEハロプロ。  
サビのフリをはじめとして目が楽しい。

# 正しい青春ってなんだろう

フリが大きくて研修生！って感じの曲。  
元気出る。

# Rainbow

実は2ndアルバム出た時はそんなに好きじゃなかった曲。  
聴きすぎて逆に好きになった。

# ミステイク（研修生ユニット）

おなじみ。  
新グループ結成が発表されてから本当に活き活きしている。  
特にまどぴ、固さが取れて表情も明るく感じる。

# 私が言う前に抱きしめなきゃね（チーム紅→たまむし）

紅は間奏のオリジナルダンスのキレが明らかに増していた。  
すごくかっこよかった。華凜ちゃん輝いてた。  
彩菜ちゃんが特に急成長してない…？  
  
たまむしももちろんかっこよかった。  
ももちゃんの歌の癖は色々言われているけど私は好き。  
えばちゃん緑似合わなくてかわいい。  
咲良ちゃんてなんか目で追っちゃうよね。

# 恋したい新党（研修生ユニット）

久しぶりに見た。恋したい新党。  
千手観音っぽいダンス、大人数でしか見たこと無かったので少人数をじっくり見ることができるのは新鮮。

# リアル☆リトル☆ガール（研修生ユニット）

北研の神曲。  
高校生組が楽しそうにかわいい曲をパフォーマンスしているのを見て癒された。

# Crying

この曲… 青春て感じですごい好き。  
コールしたい衝動に駆られる。

# Say! Hello!

この曲をやってくれるとは思わず… 歓喜。  
なんだかんだ1stアルバムでは一番好きな曲。

# アニマルランド

出た！話題のアニマルランド。  
ミュージカルっぽい。

# きみの登場

新曲！出だしから勢いがある曲。  
フリも結構激しめ。

# おへその国からこんにちは

今のメンバーでのへそこん聴きたいなーと思っていたら実現。  
冒頭セリフは葉純ちゃん。1番サビえばちゃんソロ。良い。

# 未来へ繋ぐ約束

もう終わるんだな…という雰囲気を感じ寂しくなる。

# 天まで登れ！

手拍子と手を振るフリで参加できる曲が最後。  
明るく終わってとても素晴らしい余韻に浸れた。

- 小野田華凜ちゃんの安定感
