半年ぶりのまこと。

以下、感想です。

# （1）中山夏月姫 / 涙の色【℃-ute】

去年より声がしっかり出ている。衣装も大人っぽいし綺麗。  
衣装とヘアメイクが合っているなぁ。

# （2）豫風瑠乃 / 生まれたてのBaby Love【Juice=Juice】

声の伸びもすごいし、表情も明るくて豊かで素晴らしい。  
衣装のドレスかわいい。リズムが惜しい。

# （3）橋田歩果	/ One Summer Night 〜真夏の決心〜【カントリー・ガールズ】

声きれい。気持ち良く聴ける。  
勝田さんも言ってたけど衣装かわいいね。  
ミラクルビラ配り。

# （4）植村葉純	/ Missラブ探偵【W】

大人っぽい衣装。ダンスかっこいい。  
緊張がすんごい伝わってきてめちゃ応援しちゃった。

# （5）米村姫良々 / 奇跡の香りダンス。【松浦亜弥】

歌とビジュアルの安定感はさすが。

# （6）北原もも / 都会っ子 純情（2012神聖なるVer.）【℃-ute】

声がしっかり出てるし初めてとは思えない堂々としたステージング。  
ダンスや表情がかっこいいし、歌を自分流にできている。

# （7）村越彩菜	/ 旅立ちの春が来た【スマイレージ】

明るい表情が似合うなぁ。千奈美っぽい。  
合っている選曲だし、声もちゃんと出ててすごい。

# （8）西﨑美空	/ スッペシャル ジェネレ〜ション【Berryz工房】

ハロプロっぽい衣装（チャラチャラしたシルバー）。若干つんく歌唱（好き）。  
当時のベリっぽい声でかわいい。

# （9）小野田華凜 / 「ひとりで生きられそう」って それってねえ、褒めているの？【Juice=Juice】

ダンスや表情の表現力がすごい。  
声が裏返ったけど全力の歌唱で好き。

# （10）窪田七海 / 私の魅力をわからない 鈍感な人【光井愛佳（モーニング娘。）】

セルフプロデュースが完璧。痩せたよね。すごい。  
パフォーマンスもさすがの研修生ユニット。

# （11）平山遊季 / Come with me【こぶしファクトリー】

歌うま！衣装きれい！顔面が強い。滑舌が惜しい。  
本当、今後が楽しみすぎる。

# （12）斉藤円香 / 今夜だけ浮かれたかった【つばきファクトリー】

裏返りそうなところも踏ん張って出せていた。  
表情も前よりは良くなっている。と思う。

# （13）江端妃咲 / 21時までのシンデレラ【Berryz工房】

配信トラブル発生。もう1回やったのに素晴らしいパフォーマンス。  
泣いた後らしいけど全然。明るくかわいくはきはきしていて良かった。  
追記：ハロドリ見たら1回目の方が溌剌としていた。運が悪かった…

# （14）為永幸音 / 抱いてよ！PLEASE GO ON【後藤真希】

高い赤いヒール！普段ふわふわなのに今日はCOOL。  
サビの野生感良かった。大人。

# （15）松原ユリヤ / 赤いイヤホン【アンジュルム】

歌とダンスがまた良くなっている。通る声だなぁ。  
表現力ついたらやばいよね。赤い糸出てきた。

# （16）山田苺 / SHALL WE LOVE？【ごまっとう】

衣装やダンスでスタイルが映えてた。活かしてる。  
歌声と表情の表現力がめちゃあがっている。切ない顔が良いね。

# （17）石栗奏美 / リゾナント ブルー【モーニング娘。】

強い。圧巻のパフォーマンス。  
姫良々ちゃんと窪田さんみたいにすんごい楽しんでパフォーマンスしている。  
トーク中の笑顔も良いね。

# （18）広本瑠璃 / タチアガール【スマイレージ】

自分のことわかっている。髪型、衣装、選曲が完璧。  
なんていうか、かわいい。

# まとめ

投票は格の違いを見せてくれた石栗さんに。  
（個人的趣味を優先させたら瑠乃ちゃんだったが抑えた）

**まこと賞**：小野田華凜  
**鈴木愛理賞**：中山夏月姫  
**宮崎由加賞**：北原もも  
**勝田里奈賞**：松原ユリヤ

初パフォーマンス組のレベルが高かった。  
全員伸びしろがすんごい。未来が明るい。  
特に瑠乃ちゃん、平山さんが楽しみ。いやもう全員楽しみ。  
妃咲ちゃんかわいかったし、北原さんのポテンシャルすんごいし。

来年が既に楽しみ。
